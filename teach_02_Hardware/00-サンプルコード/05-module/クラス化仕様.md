# クラス仕様（LEDController・Button・SerialPrint）

---

## LEDController — `led_module.py`

### コンストラクタ

```python
__init__(pin_no: int, freq: int = 1000, duty_max: int = 65535) -> None
```

| 引数            | 説明                                    |
| ------------- | ------------------------------------- |
| **pin\_no**   | PWM 出力に使用する GPIO 番号                   |
| **freq**      | PWM 周波数 \[Hz]（既定 1 kHz）               |
| **duty\_max** | デューティ最大値（MicroPython 既定 16 bit→65535） |

### メソッド

| メソッド                                         | 機能                                           |
| -------------------------------------------- | -------------------------------------------- |
| `on() -> None`                               | 100 % デューティで点灯                               |
| `off() -> None`                              | 0 % デューティで消灯                                 |
| `blink(times: int, interval: float) -> None` | `times` 回だけ **on → wait → off → wait** を繰り返す |
| `set_brightness(percent: float) -> None`     | 0–100 % をデューティ値に換算して設定                       |
| `set_freq(freq: int) -> None`                | PWM 周波数を変更                                   |

> **メモ**: Raspberry Pi Pico のオンボード LED (GP25) も PWM 対応。

## Button — `button_module.py`

### コンストラクタ

```python
__init__(pin_no: int, pull: str = 'UP') -> None
```

| 引数          | 説明                             |
| ----------- | ------------------------------ |
| **pin\_no** | ボタン接続 GPIO 番号                  |
| **pull**    | `'UP'` or `'DOWN'` — 内部プル抵抗の向き |

### メソッド

| メソッド                                             | 機能                     |                      |
| ------------------------------------------------ | ---------------------- | -------------------- |
| `is_pressed() -> bool`                           | 押下状態を即時に返す             |                      |
| \`wait\_for\_press(timeout: float                | None = None) -> bool\` | 押下または `timeout` まで待機 |
| `on_press(callback: Callable[[], None]) -> None` | 割込みハンドラ登録（イベント駆動）      |                      |

---

## SerialPrint — `print_module.py`

### コンストラクタ

```python
__init__(uart_id: int = 0, baudrate: int = 115200) -> None
```

| 引数           | 説明                         |
| ------------ | -------------------------- |
| **uart\_id** | 使用 UART 番号（Pico は 0 または 1） |
| **baudrate** | ボーレート \[bps]（既定 115 200）   |

### メソッド

| メソッド                                                    | 機能                    |
| ------------------------------------------------------- | --------------------- |
| `print(*args, sep: str = ' ', end: str = '\n') -> None` | `print()` 互換で UART 出力 |
| `println(msg: str) -> None`                             | 末尾改行付きで送信             |

---

> **学習のヒント**
>
> * LED 明るさ制御は PWM デューティと 16bit スケール（0–65535）の相関を体験すると理解が深まる。
> * `Button.on_press()` で割込み駆動版を実装し、非同期処理との違いを確認してみよう。
> * `SerialPrint` は標準 `print()` を UART にリダイレクトするイメージで実装すれば扱いやすい。
