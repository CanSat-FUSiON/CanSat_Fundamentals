# クラス一覧と仕様

## LEDController (led_module.py)

クラス `LEDController` は指定したGPIOピンでLEDを制御します。

### コンストラクタ

`__init__(self, 'pin_no': int) -> None`

- `pin_no`: LED接続GPIOピン番号
- 指定ピンを出力モードに設定
- 戻り値: None

### メソッド

#### `on(self) -> None`

- LEDを点灯します
- 引数なし
- 戻り値: None

#### `off(self) -> None`

- LEDを消灯します
- 引数なし
- 戻り値: None

#### `blink(self, 'times': int, 'interval': float) -> None`

- `times`: 点滅回数 (1以上)
- `interval`: 点灯/消灯間の待機時間 (秒, 0.0以上)
- 各サイクルは `on()` → `off()`
- 戻り値: None

## SerialPrint (print_module.py)

クラス `SerialPrint` はUARTでフォーマット済みテキストを送信します。

### コンストラクタ

`__init__(self, 'uart_id': int = 0, 'baudrate': int = 115200) -> None`

- `uart_id`: 使用するUART番号
- `baudrate`: 通信速度 (bps)
- 指定UARTを設定
- 戻り値: None

### メソッド

#### `print(self, *'args', 'sep': str = ' ', 'end': str = '\n') -> None`

- *`args`: 出力対象の値
- `sep`: 引数間の区切り文字
- `end`: メッセージ末尾に追加する文字列
- 複数値を連結して1つのシリアルメッセージとして送信
- 戻り値: None

#### `println(self, 'msg': str) -> None`

- `msg`: 送信する文字列
- メッセージ末尾に改行を追加して送信
- 戻り値: None

## BrightnessPWM (pwm_module.py)

クラス `BrightnessPWM` はPWM信号でデバイスの明るさを調整します。

### コンストラクタ

`__init__(self, 'pin_no': int, 'freq': int = 1000, 'duty_max': int = 1023) -> None`

- `pin_no`: PWM出力GPIOピン番号
- `freq`: 周波数 (Hz)
- `duty_max`: 最大デューティ値
- 指定ピンでPWMを初期化
- 戻り値: None

### メソッド

#### `set_duty(self, 'duty': int) -> None`

- `duty`: `0`～`duty_max` のデューティ値
- デューティを直接設定
- 戻り値: None

#### `set_brightness(self, 'percent': float) -> None`

- `percent`: 明るさ (%) (0.0～100.0)
- 内部でデューティ値へ変換して設定
- 戻り値: None

#### `set_freq(self, 'freq': int) -> None`

- `freq`: 新しい周波数 (Hz, >0)
- PWM周波数を変更
- 戻り値: None

## Button (button_module.py)

クラス `Button` はGPIO入力でボタン押下を検出し、コールバックも登録できます。

### コンストラクタ

`__init__(self, 'pin_no': int, 'pull': str = 'UP') -> None`

- `pin_no`: ボタン接続GPIOピン番号
- `pull`: `UP` or `DOWN` (プル抵抗設定)
- 指定ピンを入力モードかつプル設定で初期化
- 戻り値: None

### メソッド

#### `is_pressed(self) -> bool`

- ボタンが現在押されているか判定
- 引数なし
- 戻り値:
    - True: 押下中
    - False: 非押下

#### `wait_for_press(self, 'timeout': Optional[float] = None) -> bool`

- `timeout`: 待機上限 (秒)、Noneで無限待機
- 押下またはタイムアウトまで待機
- 戻り値:
    - True: 押下された
    - False: タイムアウト

#### `on_press(self, 'callback': Callable[[], None]) -> None`

- `callback`: 引数なしのコールバック関数
- 押下時に非同期で呼び出される
- 戻り値: None
